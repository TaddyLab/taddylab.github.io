Variable_Name Format
household_id INT   Merges with "household_id, purchase_date, store_name_id" in "Purchases"
purchase_date DATE (YYYY-MM-DD)
store_name_id integer
total_spent float
tdlinx_code integer

in the transaction files. The tdlinx_code, however, can only be
obtained from these files.
