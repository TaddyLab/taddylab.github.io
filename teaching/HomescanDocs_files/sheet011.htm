(text), e.g style and style_descr. Multi is the only exception. It
has a single numeric value multi.

NOTE3: The format of the value of size1_descr and size2_desc is:
NUMBER UNITS; the number is a real number, the units are denoted by 4
letters, e.g "16.0 MLOZ"

products sold in 2007.
